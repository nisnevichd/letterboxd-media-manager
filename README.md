# Letterboxd Media Manager

A Java project to track books and movies, including ratings, reviews, and completion status.

## Features
- Add books and movies
- Mark items as completed (read/watched)
- Add ratings and reviews
- Display item information

## Concepts Used
- Classes and Objects
- Inheritance
- Abstract Classes
- Method Overriding
- Encapsulation

## Files
- MediaItem.java (abstract parent class)
- Book.java
- Movie.java
- Main.java

## Notes
This project is being built over multiple days to practice core Java concepts for exams.
